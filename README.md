# GDT3CR - Ethical Hacking

This repository contains laboratory work, technical reports, and practical demonstrations completed as part of the course **Ethical Hacking (GDT3CR)**.

The course focuses on understanding attacker methodologies, performing ethical penetration testing, analyzing intrusions, and evaluating IT security from technical, ethical, and societal perspectives.

---

## 🎯 Learning Outcomes

Upon completion of the course, the student is able to:

### 1. Explain Attacker Methodologies
Describe fundamental principles and techniques used by attackers to:
- Gain unauthorized access to systems
- Escalate privileges
- Maintain persistence
- Conceal traces of intrusion

---

### 2. Perform Ethical Penetration Testing
Apply acquired knowledge in practical scenarios by:
- Conducting structured penetration tests (pen-tests)
- Using industry-standard tools
- Documenting findings professionally
- Following ethical and legal guidelines

---

### 3. Analyze Security Breaches
Perform analysis of:
- System compromises
- Intrusion indicators
- Log data and attack traces
- Information security risks

---

### 4. Evaluate IT Security Solutions
Assess strengths and weaknesses of:
- Operating systems
- Network architectures
- Authentication systems
- Security mechanisms and configurations

With regard to confidentiality, integrity, and availability (CIA triad).

---

### 5. Demonstrate and Present Security Concepts
Independently:
- Present penetration testing techniques
- Demonstrate attack simulations in educational contexts
- Explain technical findings clearly and responsibly

---

### 6. Reflect on the Societal Role of IT Attackers
Critically evaluate the societal role of IT attackers from:
- A social perspective
- An ethical perspective
- An economic perspective

Including discussions about:
- Responsible disclosure
- Cybercrime
- Defensive security research
- The balance between offense and defense

---

## 🛠 Topics Covered

- Linux system analysis
- Network service enumeration
- Port scanning
- SMB / SSHFS / remote mounting
- Kernel exploration
- Netcat and shell management
- TFTP and lightweight transfer protocols
- SSH and remote administration
- Intrusion simulation
- Security risk evaluation
